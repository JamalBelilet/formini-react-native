# NativeBase
An open source framework to build React Native apps
over a single JavaScript codebase for Android and iOS.

1. installation of NativeBase
> npm i --save native-base

2. install expo fonts
> expo install expo-font

3. load fonts (within useEffect of the root element ex: App.js)
```JSX
import * as Font from 'expo-font';
import { Ionicons } from '@expo/vector-icons';

useEffect(async () => {
  await Font.loadAsync({
    Roboto: require('native-base/Fonts/Roboto.ttf'),
    Roboto_medium: require('native-base/Fonts/Roboto_medium.ttf'),
    ...Ionicons.font,
  })
}, [])
```
